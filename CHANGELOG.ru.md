- Первый релиз: проверка обоснованности, верификация цитат, анализатор чанкинга, аудитор retrieval

- Рендер отчётов в text, JSON и JUnit XML
- CLI, объединяющий команды check / audit-retrieval / regression
- Набор юнит-тестов для каждого модуля
